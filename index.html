PONKEY

Web apps — Edge / Chrome で開いてください
SYNC
複数台の PONKEY に同じ BPM を送ってテンポを揃える。「同期」で頭出し(同時スタート)。
要 Web Bluetooth(Edge / Chrome)
SEQUENCE
ソングモード。スロット(1小節ループ)を編集し、シーケンスに並べて演奏。本体と双方向同期。
要 Web Bluetooth(Edge / Chrome)
TRANCE
PONKEY と Logic / Ableton 等を MIDI で連携(多台同期・クロック)。
要 Web MIDI(Edge / Chrome)
Safari / Firefox は Web Bluetooth / Web MIDI 非対応です。
PONKEY — pocket-sized MIDI instrument.
